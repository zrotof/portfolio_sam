Description Le restaurant CASABREIZH est un restaurant situé dans la ville de Rennes. Il propose des plats africains et des plats bretons. Il possède un site web vieux de plus de 10 ans, mais ce dernier ne répondait plus aux réalités et exigences des clients. Et aussi, il est impossible pour le restaurateur de mettre à jour ses menus ou indiquer aux utilisateurs l'ndisponibilité ou non des plats. Les exigences de ce projet sont les suivantes :
  • permettre aux utilisateurs de consulter les menus, plats, desserts, boissons proposés par le restaurant
  • créer une interface administrateur qui permet de mettre à jour les plats proposés sur le site
  • optimiser le référencement SEO
  • terminer le développement du site web et le mettre en ligne
Technologies utilisées
Angular 12
Node js
Express js
Mongo DB
OVH
JSON Web Token
Adobe XD
