Description SSCOVID19 est un projet personnel que j'ai réalisé dans le but de participer activement à la lutte contre la covid-19 dans le monde. Il offre la possibilité à de rester informé sur l'évolution du virus notamment avec des mises à jour automatiques toutes les 45 minutes. C'est une application web bilingue ( anglais, français ) qui prend en compte des données de plus de 215 pays et qui permet:
  • de consulter les chiffres totaux mondiaux de l'épidémie
  • de consulter les chiffres par continents et par pays
  • de visualiser les graphes d'évolution de la maladie et les graphes d'évolution des vaccinations ( depuis janvier 2020 ) dans le monde et par pays
  • de visualiser la carte du monde avec des indicateurs de couleur suivant la gravité des cas dans différents pays
  • de rester à jour sur les faits d'actualité de cette pandémie
Technologies utilisées
Angular 11
Node js
Express js
PostgreSQL
AmCharts
Google Analytics
Heroku
Hostinger
